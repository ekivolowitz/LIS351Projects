What is Ethereum?


Ethereum is described as Ethereum is a decentralized platform that runs smart contracts: applications that run exactly as programmed without any possibility of downtime, censorship, fraud or third-party interference.

How Does it Work?

At a very high level, Ethereum is an application of Blockchain, a Distributed Ledger Technology. Blockchain allows for a network of nodes receiving transactions from eachother, and validating them. When a transaction is validated on one node, it's sent to other nodes for validation. Thus, Blockchain allows for a decentralized form of trust that doesn't rely on a single third party entity.

How is This Different Than Bitcoin?

By How Does it Work section alone, there is no difference between Ethereum and Bitcoin. In reality there are many differences. To me the biggest difference is the concept of a smart contract. Code can be written to the Blockchain, and can be executed by anyone. That code is ran on each node, producing the same results everywhere, which turns the Ethereum Blockchain into the world's largest, most distributed computer ever.
